## Project Overview

DriveFleet Server is the backend API service for the DriveFleet car rental platform. It handles car management, booking operations, user-related APIs, and MongoDB database integration.

## Features

- RESTful API for car rental platform
- MongoDB database integration
- CRUD operations for cars
- Booking management system
- Search and filtering API support
- Environment variable secured configuration
- CORS enabled for frontend integration
- Deployed on Render

## Technologies Used

- Node.js
- Express.js
- MongoDB
- dotenv
- cors
